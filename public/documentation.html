Movie API Documentation

This will be an api for calling movie data.

Purpose URL HTTP Method Request body data format Response body data format
Get all Chicken Breeds /breeds GET None A JSON object holding data about all chicken breeds
Get Chicken Breed by name /breeds/[name] GET None A JSON object holding data about a single breed containing an ID, Name, APA Class, Egg Color, and Description.
{
id:1,
name:Ameraucana Black,
apa class:AOCCL(All Other Combs Clean Leg),
egg:
{
color:blue/green
size:small
production:good
},
Comb type:pea
}
Get all breeds by egg color /breeds/eggs/[color] GET none A JSON object holding all breeds with specified color eggs
Get all breeds by APA Class /breeds/class/[class] GET None A JSON object holding all breeds with specified class
Allow a new user to register /users POST A JSON object holding the new users Username, password, and email. Optional: name.
{
username:JaneDoe42,
password:small toaster horse leaf2,
email:janedoe@janedoe.com
} 		A JSON object holding the new users information
Allow user to update their user info /users/[username] PUT A JSON object holding user data to be changed A JSON object holding updated user information
Allow user to view all breeds on their favorites list /users/[username]/favorites GET None A JSON object holding users list of favorite breeds.
Allow user to add breeds to their favorites list /users/[username]/favorites/[breed] POST None A text message indicating that the breed was added to their favorites list successfully.
Allow users to remove breeds from their favorites list /users/[username]/favorites/[breed] DELETE None A text message indicating that the breed was deleted from their favorites list.
Allow user to delete their account /users/[username] DELETE None A text message indicating that the users account has been deleted.
